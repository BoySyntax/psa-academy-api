# PSA Academy API

## Overview
PHP REST API for PSA Academy educational platform.

## Deployment
This API is configured to run on Railway using PHP's built-in web server.

## Environment Variables
- `MYSQL_URL` - MySQL database connection string
- `DATABASE_URL` - Alternative database connection string

## Endpoints
- `/student` - Student management
- `/teacher` - Teacher management
- `/admin` - Admin functions
- `/management` - Management functions

## Database
Connects to Railway MySQL service with automatic configuration.
